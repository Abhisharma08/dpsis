# **App Name**: DPSIS Bridge to Success

## Core Features:

- Value Proposition Display: Prominent display of the program's value proposition (seamless transition to Primary 2).
- Contact Form: An embedded contact form.
- FAQ Section: A FAQ section addressing common concerns about the Bridge Program.
- Show parent testimonial: Show parent testimonial
- Quick Links: Links to School Website, School Fee, ECA and CCA, School facilities, Cambridge Curriculum, and ICSE Curriculum

## Style Guidelines:

- Primary color: Teal (#008080) to evoke trust, growth, and stability in education.
- Background color: Light gray (#E0E0E0), almost white, to create a clean, spacious, and trustworthy backdrop that directs focus towards the teal elements, while maintaining an accessible and friendly atmosphere.
- Accent color: Orange (#D4A274) to draw attention to important details and create a sense of vibrancy.
- Body and headline font: 'Alegreya', a serif font that has an elegant, intellectual, contemporary feel.
- Use a clear, sectioned layout with ample white space for readability.
- Use friendly and educational icons related to Bridge Program and DPSIS values.
- Subtle, child-friendly animations on images to increase engagement.
